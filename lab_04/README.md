# Build and run

## Files

```sh
$ python3 main.py --file [input file] --out [output file] [-p {P param}] [-q {q param}] --seed [number for random]
$ python3 main.py --file [input file] --out [output file] [-p {P param}] [-q {q param}] --seed [number for random] --decode
```

## Message

```sh
$ python3 main.py --message [message] [-p {P param}] [-q {q param}] --seed [number for random]
$ python3 main.py --message [message] [-p {P param}] [-q {q param}] --seed [number for random] --decode
```
